Project 1 Elevator Panel Redesign

To get a C, you need to:


✔ Find an elevator in the apartment/hotel/airport. Only one example for each student.
✔ Take photos of the control interface
✔ Show a gif image of the control interface in operation.
✔ Find the issues with the current design.
✔ Explain why it is good/bad.

Good/Bad Explained

This elevator panel design is good as it is straightforward, easy to navigate, and accessible to a diverse group of people, including the disabled.
On the other hand, the alarm button has a red circle around it as if the button was activated like the floor buttons, which could confuse the user.

To get a B, you need to:


✔ Think about the common things that you use an elevator. List your most common uses and other more rare uses. Does the interface make doing those common things easier?
✔ Think about how the user interacts with the elevator. What is the common sequence of actions?
✔ How does the elevator support the user to figure out how to make it work?
✔ How does the elevator provide feedback to the user?
✔ What are some common mistakes you can make with this current design?
✔ Suggest the improvements to make on the control interface. Sketch your solution. And justify your design decisions.
